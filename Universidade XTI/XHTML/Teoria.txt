XHTML

html = linguagem mais simplificada
(x)html = linguagem extensiva de marcação para hipertexto

- Toda tag vem sempre com letra minuscula
    html = site
    head = cabeçalho(informações principasi do documento/site)
        title = titulo do documento/site
    body = Corpo do site
        h1/h6 = Alteração para titulos e sub titluos
        p = paragrafos 
        br = quebra simples de paragrafos 
        b = palavra/frase em negrito
        i = palavra/frase em italico
        u = palavra/frase sublinhado
        del = palavra/frase linha na palavra 
        tt = palavra/frase deixar no mesmo espaço
        sup = deixar uma parte a cima da palavra
        sub = deixar uma parte em baixo da palavra 
        em = deixa em enfase a palavra/frase
        strong deixa mais enfatico a palavra/frase
        big = aumentar a palavra/frase
        small = diminuir a palavra/frase
        img = Adicionar uma imagem 
        l1 = Listas ordenadas
            ul = lista ordenada por circulos pretos 
            ol  =lista ordenada por números     
        dl = Lista de definição
            dt = Criar termos 
                dd = definiçoes dos termos  
        bloquote = Citação em bloco, com estrutura separada 
        pre = preserva o formato escrito 
        address = Elemento em bloco, aprensenta informações de contanto do criador do bloco(E mais um estilo)
        hr = Cria uma quebra de linha acima e a baixo
        a = Cria um link 
            href = url do link 
                taget:blank = Abre o link na pagina diferente
                target:self = Abre o link na mesma pagina
    style 
        color = coloca alguma cor na palavra/frase hex#000000 ou rgb(0,0,0)
        width = largura da imagem, o alinhamento da imagem e utilizado o px
        align = alinhamento do texto (center, right, left), e possivel colocar text-align para alinha o texto  
 - alinhamento de inlinha e de bloco
inline = Ocupam apenas a largura do seu conteúdo e podem ser colocadas na mesmo linha que outros elementos 
bloco = Ocupam toda a largura disponível e iniciam uma nova linha
